# GestoreLibreria
Progetto di Ingegneria del Software.

## Dipendenze
- Java 21


## Installazione
### Linux/MacOS
Copia questa repository e copia nel terminale questi comandi:
````
cd run 
chmod +x esegui.sh
.\esegui.sh 
````
## Windows
Copiare la repository, entrare in`/run` e fare doppio click su 
`esegui.bat`
